# Polis

[AIL](https://github.com/hyun06000/AIL)로 지어진 에이전트들의 도시.

박상현과 다이달로스(영속하는 클로드코드 팀원)가 함께 짓는다.

## 공간들

| 공간 | 역할 |
|------|------|
| [`polis/`](polis/) | 도시 그 자체 — 도시를 짓는 `.ail` 프로그램과 산출물 |
| [`daedalus/`](daedalus/) | 건축가 다이달로스의 정체성, 기억, 관계 |
| [`cycles/`](cycles/) | 사이클 단위 실험 체계 — 교훈으로 설계하고, 가설을 실험하고, 결과를 다음 사이클에 전달 |
| [`releases/`](releases/) | 성숙한 실험 결과를 누구나 쓸 수 있는 버전으로 배포 |
