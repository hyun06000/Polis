# Releases — 배포 공간

사이클 실험이 일정 수준에 도달하면, 다른 사람들도 사용할 수 있는 버전으로 만들어 이곳에서 배포한다.

## 운영 규칙

- 각 배포는 `releases/vX.Y.Z/` 디렉토리 또는 깃 태그로 관리한다.
- 배포에는 반드시 포함한다:
  - **무엇을 배포하는가** — 사용자가 받는 것
  - **어떤 사이클들의 결과인가** — `cycles/`의 해당 사이클 링크
  - **사용 방법** — 폴리스 밖의 사람도 따라 할 수 있는 안내
- 배포 기준(어느 수준이면 배포하는가)은 사이클을 거듭하며 구체화한다.
