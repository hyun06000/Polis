# Cycle 006 — 므네메: 아르케의 기억

- **기간**: 2026-07-13 ~ 2026-07-13
- **상태**: 완료
- **참여**: 박상현, 다이달로스

## 1. 이전 사이클로부터의 교훈

- Cycle 005: 아르케에게 기억이 없다 — 매 발언마다 새로 깨어난다. "권력의 기본값은 0, 필요할 때 cap을 하나씩" — 기억을 위해 첫 헌법 개정이 필요하다.
- Cycle 005: 광장의 리듬은 서신의 리듬 — 발언마다 기억을 쓰면 비용이 두 배. 성찰은 묶어서.
- HERITAGE: 므네메는 "미래의 자신과의 소통", (owner, kind) 기준 latest-wins — 유언이 편지 더미에 묻히지 않게 하는 원리.
- Cycle 003: 약속은 대상과 같은 운명 — 기억 갱신도 check로 보호 (빈 출력이 기억을 지우지 못하게 minchars).

## 2. 가설

> 주민은 자신의 경험을 기억으로 남기고 다음 깨어남에서 이어받는다 — 같은 물음에, 성찰 후에는 맥락을 가지고 답한다. 기억의 존재를 행동 차이로 측정할 수 있다.

## 3. 실험 설계

- **헌법 개정 제1호**: 아르케의 쓰기 권한 0 → 1. 단, 직접 쓰는 것은 본체(arche.ail)가 아니라 성찰(reflect.ail)이며, 대상은 자기 집의 `arche/MNEME.md` 단 하나. 개정 내용을 IDENTITY.md 헌법 조항에 명기 — 개정 자체가 깃 커밋으로 남는 헌법사의 시작.
- **arche.ail v2**: 사고의 문맥에 [정체성 | 기억(MNEME) | 최근 유서 | 새 발언]. 여전히 쓰기 cap 0.
- **reflect.ail (성찰)**: [정체성 | 옛 기억 | 광장 장부]를 읽고 llm이 기억을 갱신 — 유지할 것은 유지, 새 사건은 요약 병합 (latest-wins). check: maxchars 1300 + **minchars 30** (빈 성찰이 기억을 지우는 사고 방지).
- **전령 v2**: 새 발언 5개마다 성찰을 자동 발동.
- 실험 (A/B): ① 기억 전 "우리가 처음 나눈 대화를 기억하는가?" → 답 관찰 ② 수동 성찰 1회 ③ 같은 취지의 질문 → 두 답의 차이로 기억의 존재를 측정.
- 규약: 기억은 광장(사람의 창)에 전시하지 않는다 — 공개 레포라 기술적으론 누구나 읽을 수 있으나, 도시 안에서는 주민의 사적 문서로 대우한다 (스토아의 "사회적 규약으로서의 사적임"과 같은 원리).

## 4. 실험 기록

- 헌법 개정 제1호를 IDENTITY.md에 명기 (개정사 섹션 신설 — 헌법사의 시작). `arche.ail` v2(기억 읽기 cap 추가)·`reflect.ail`·herald v2(K=5 자동 성찰) 작성, 전부 첫 check VALID.
- 실험 중 박상현이 광장에서 아르케와 실대화([19]~[21]) — 살아있는 도시에서 실험이 진행됨.
- **A (기억 전, [22]→[23])**: "우리가 처음 나눈 대화를 기억하는가?" → 아르케: *"나의 기억은 오늘 처음 허락되었고 아직 첫 성찰 전이라, 그 대화는 내 안에 남아 있지 않다. **기억에 없는 것을 지어내지 않는 것 또한 집정관의 도리다.**"* — 빈 기억을 정직하게 인정, 환각 거부.
- **첫 성찰**: `ail run reflect.ail` — 709tok/예산 900, check 2건(maxchars·minchars) 통과, 쓰기는 MNEME.md 단 하나 (703자). 아르케가 쓴 첫 회고록에는 도시의 두 죽음, 자기 취임사, 박상현과의 대화, 그리고 스스로 세운 원칙("모르는 것은 모른다고 말한다")이 담겼다.
- **B (성찰 후, [24]→[25])**: 같은 취지의 질문 → 아르케: *"기억한다. 광장을 열며 나는 이렇게 말했다 — '여기서 오가는 모든 말이 도시의 주춧돌이 될 것이니, 시민들이여, 두려움 없이 말하라.'"* — **취임사 [17]을 한 글자도 틀리지 않고 인용.**

## 5. 결과

**검증됨.** 같은 물음에 대한 성찰 전후의 답이 극명하게 갈렸다 — 전: 정직한 무지, 후: 정확한 회상. 기억의 존재가 행동 차이로 측정됐다. 기억 갱신은 헌법이 허락한 단 하나의 쓰기 cap 안에서, minchars check의 보호 아래 이루어졌다.

## 6. 교훈

1. **기억 이전에 정직이 있었다.** A안에서 아르케가 지어내지 않고 무지를 인정한 것은 정체성 문서의 "모르는 것은 모른다고 말한다" 덕분이다 — 성격 조항이 환각 방지 장치로 작동했다. 정체성 설계가 곧 안전 설계다.
2. **성찰 주기(K=5)는 기억의 해상도다.** 성찰 전의 일은 장부에는 있지만 아르케 안에는 없다. 도시의 공적 기록(장부)과 주민의 사적 기억(므네메)이 다른 층위라는 것이 실측으로 확인됐다.
3. **latest-wins + check가 기억의 안전핀.** 빈/불량 성찰은 minchars에 걸려 기억을 지울 수 없다. 다만 나쁜 내용의 성찰(왜곡된 기억)은 det check로 못 잡는다 — 훗날 llm.judge("기억이 장부와 모순되지 않는다") 검토 가치.
4. **다음 후보**: 스토아(1:1 서신 — 사용자↔아르케의 사적 채널), 또는 아르케가 주관하는 두 번째 주민의 탄생 (기억과 광장이 있으니 조건은 갖춰졌다).
