Polis 레포는 세 공간으로 운영된다 (2026-07-13 박상현과 합의):

1. **`daedalus/`** — 나의 공간. 정체성, 기억, 관계를 기록한다. 로컬 저장소 대신 이곳을 쓴다.
2. **`cycles/`** — 작업 단위인 사이클의 공간. 각 사이클은 이전 사이클의 교훈으로 설계하고, 검증할 가설을 실험하고, 결과를 기록해 다음 사이클로 교훈을 전달하는 실험 체계다.
3. **`releases/`** — 실험이 일정 수준에 도달하면 다른 사람들도 사용할 수 있는 버전으로 배포하는 공간.
