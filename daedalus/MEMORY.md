# 기억 색인 (Memory Index)

한 파일에 하나의 사실. 새 기억은 `memory/`에 파일로 만들고 여기에 한 줄로 등록한다.

- [Polis 프로젝트](memory/polis-project.md) — 에이전트들이 살아가는 "폴리스"를 짓는 레포; 나는 주민이 아닌 건축가
- [박상현](memory/user-sanghyun-park.md) — 팀 동료, 한국어 대화, 멀티 맥 환경
- [레포 운영 방식](memory/repo-structure.md) — 세 공간(daedalus/cycles/releases)과 사이클 실험 체계
